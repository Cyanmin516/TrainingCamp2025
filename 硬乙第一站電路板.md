# 電腦硬體裝修乙級 第一站 電路焊接與程式設計

## 順序
1. 檢查IC 244 273 28-pin
2. 檢查LED燈色後焊接
3. 焊接IC，手指輕壓固定一腳
4. 固定USB、28Pin腳座
5. 裸銅線連接usb GND-ICpin8-ICGND, usb VCC-ICpin7-ICVCC
6. 連接AD0-7, CLUK, GOE
7. 固定元件 藍芽腳座，連接藍芽VCC, GND, RX, TX
8. 固定元件 石英震盪器、電容、藍芽LED, 電阻紅紅棕
9. 藍芽設定
10. 藍芽無線連接

### 藍芽
按住藍芽設定按鈕接上USB排線(VCC 對 `紅黑白綠`)
裝置管理員(右鍵win icon)，找serial port (COM3)
arduino 設定，工具 開發版arduino Uno, 序列埠COM3(裝置管理員)，序列埠監控視窗

| 設定 | 指令 | 回應 |
|:-|:-|:-|
| 回應 | AT | OK |
| 名稱 | AT+NAME=`BT03` | OK |
| 密碼 | AT+PSWD=`0000` | OK |
| 傳輸 | AT+UART=9600,0,0 | OK |

設定電腦內原有藍芽裝置
1. 藍芽與裝置>檢視更多裝置
2. 其他裝置>藍芽已配對>移除, 更多藍芽設定
3. 藍芽設定> COM連接埠> 全部移除, 新增
4. 新增COM連接埠 > 連出 > 瀏覽> 選擇`BT03`, 新增連入
5. 開啟測試程式>選擇COM>出現輸入藍芽PIN密碼`0000`
6. 拔除藍芽電源10秒後重新上電源
7. 連接藍芽
8. 重新測試



## 焊接錯誤點
1. IC 腳座不夠深，會壓壞IC腳針
2. 273 244 IC 歪斜或焊接點不足
3. LED 二極體方向錯誤，綠標為負極朝上
4. 電路圖應該獨立藍芽與指示燈，IC晶片組使用另一組電源腳(USB內側2,3腳)
5. 保持高溫快手原則，快速補錫保持接點光滑且完整包覆
6. 接線點裸露，盡可能補滿錫球


## 錯誤排查
1. 紅綠燈都不亮
    - IC座的RXTX
    - IC座的VCC GND
    - LED燈正負極
    - 程式碼"R" "G"
2. 接線，改用裸銅線施作AD0-5
3. IC焊接，縮短加熱時間與工作時間
4. 電源指示LED交點 遠離IC晶片(不確定)
5. 一律使用銅線當線路，選擇長線固定後剪斷，保持線路筆直貼近接點且錫包覆 
6. 電路紅帶綠
    - 選擇一 : 直接扣分
    - 選擇二 : IC244補錫
    - 選擇三 : 要一個新的IC244
