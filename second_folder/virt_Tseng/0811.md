### 程式可讀性
- readable要求，變數宣告要清楚，`action`動作不要縮減成`a`
    - Label = txt
    - Button = btn, btnRed, btnGreen, btnSPort
- boundary要求，r(8) => r(9), 八個動作加結束 


#### VB GUI
- Windows Form
- OvalShape 圓形 16
- Label 標籤 1
- Button 按鈕 4
- Timer 時鐘 1
- Serial Port 序列埠 1

**Timer**
- enable = `True`
- interval = `1000`

**OvalShape**
- size = `30,30`
- FillType = `solid`

**SerialPort**
vm close and add serial port

```c
G1= green 0000 0001
G4= green 0000 0100
G72= green 0111 0010
G80= green 1000 0000


vb 十六進位表示 &HC0
C# 十六進位表示 0x7F
```
