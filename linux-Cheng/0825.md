## UEFI
查看硬體資訊
- DRAM 記憶體 數量及大小
- SATA 匯流排 硬體或光碟機 數量及大小
- Boot Priority 開機順序 ，滑鼠拖曳順序，重新安裝要改成USB開機
    - 硬碟(SSD) : 內部裝置
    - 光碟/USB : 外接裝置
    - 網路 : 網路啟動
- `F10` : save and exit
- VT-d : 虛擬化功能 

## 硬體裝修
1. 現場指定內容表背面空白可以做筆記
2. 標籤色貼紙，例如SSD硬碟與SATA線，五公分貼在交接處，拔開一半在硬碟上一半在SATA線上。
