## 模擬練習
1. 準備兩台測驗用電腦
2. 還原主電腦
3. 由監評埋藏故障點
4. 排除錯誤後完成命題


### 故障排除
- Q 接外顯不亮
- A 插頭鬆脫
-   卡曹鬆脫
-   BIOS 設定跑掉 : advanced, 內建裝置設定, DVI port audio, disable=>enable
-   還原BIOS

- Q 錯誤蜂鳴器
    三短    : Dram
    五短    : CPU
    六短    : 鍵盤
    一長八短: 顯示卡或螢幕
