## 絞線
為甚麼要絞線
1. 抗電磁干擾
2. 抗拉扯 
    同一長度絞線需要更長

+ 對絞
    + 兩條線絞線
+ 星絞
    + 四對線絞線
    + 色碼 藍黃綠紅紫(主色碼)
    + 藍白棕黑1p2p、黃白棕黑3p4p、綠白棕黑5p6p、紅白棕黑7p8p、紫白棕黑9p10p

```
實務上 20公分退絞線
P單位 = 兩條一對
4條線 = 2p
2rd P = 藍白的棕黑
10th P= 紫白的棕黑
目前實作 100p 線 ，實務上多在200p以下
600p 為 超級簇 
弱電在220V以下的線路
```
