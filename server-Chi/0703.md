Linux:
不同版本差異，目錄，

需要備份的資料夾
/etc

usb 掛上去，圖形介面
/media
手動掛載
/mnt

/run
第三方軟體、PID、PORT


l linkfile 捷徑
d directory目錄/資料夾

檢測容量
ll -d /proc
ll -d /sys
以上兩個都會是0，因為都是映像RAM

看說明
man ll

看指令位置
which ll

查看樹系
df

tmpfs檔案
/dev/shm
把檔案放進 /dev/shm 修改較快

pwd 現在目錄

`.` : 目前目錄下
`..`: 上一層目錄
`~` : 使用者家目錄(desktop、download...)
`/` : 管理者根目錄(etc, var, sys...)
